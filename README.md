# Flying Skate Website

Static GitHub Pages site for App Store privacy/support URLs.

Recommended GitHub Pages settings:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/`

Expected URLs after publishing:

- Main page: `https://leftyrugiero.github.io/flying-skate-site/`
- Privacy Policy: `https://leftyrugiero.github.io/flying-skate-site/privacy.html`
- Support: `https://leftyrugiero.github.io/flying-skate-site/support.html`

Update the App Store button in `index.html` once the public App Store URL exists.
